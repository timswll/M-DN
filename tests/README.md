# Tests

Dieser Ordner bündelt die Projekt-Tests getrennt vom Produktivcode.

## Struktur

- `server/`: Unit-Tests für die Backend-Logik
- `server/helpers/`: kleine Test-Utilities und wiederverwendbare Fixtures

## Ausführen

```bash
npm test
```

Für kontinuierliches Testen beim Entwickeln:

```bash
npm run test:watch
```
