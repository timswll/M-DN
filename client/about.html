📖 Spielregeln

Ziel des Spiels

Jeder Spieler muss seine vier Spielfiguren von der Startposition aus über das Spielfeld in die eigenen Zielfelder (das „Haus") bringen. Wer als Erster alle vier Figuren im Haus hat, gewinnt.

Spielvorbereitung

Es können 2 bis 4 Spieler teilnehmen. Jeder Spieler bekommt vier Figuren in einer Farbe (Rot, Blau, Grün oder Gelb), die zu Beginn auf den eigenen Startfeldern stehen.

Spielablauf

  1. Die Spieler würfeln reihum. Wer dran ist, würfelt einmal.
  2. Figur ins Spiel bringen: Eine Figur kann nur mit einer gewürfelten 6 auf das Startfeld gestellt werden. Solange alle Figuren im Haus stehen, darf der Spieler bis zu dreimal würfeln.
  3. Ziehen: Die gewürfelte Augenzahl bestimmt, wie viele Felder eine Figur vorrückt.
  4. Schlagen: Landet eine Figur auf einem Feld, das bereits von einer gegnerischen Figur besetzt ist, wird die gegnerische Figur zurück auf ihre Startposition geschickt.
  5. Sechs gewürfelt: Wer eine 6 würfelt, darf nach seinem Zug erneut würfeln.
  6. Zielfelder: Die Zielfelder können nur über den eigenen Eingang betreten werden. Im Zielbereich darf nicht übersprungen werden.

Spielende

Das Spiel endet, sobald ein Spieler alle vier Figuren sicher in die eigenen Zielfelder gebracht hat. Dieser Spieler gewinnt das Spiel.

ℹ️ Über dieses Projekt

Diese Online-Version von Mensch Ärgere Dich Nicht wurde als Web-Projekt entwickelt. Das Spiel nutzt moderne Web-Technologien:

  • Frontend: HTML5, CSS3 (Container Queries, Custom Properties), Vanilla JavaScript
  • Backend: Node.js mit Express und Socket.io für Echtzeit-Kommunikation
  • Responsive Design: Spielbar auf Desktop, Tablet und Smartphone

Das Spiel unterstützt 2–4 Spieler, die gleichzeitig über das Internet miteinander spielen können. Erstelle einfach ein Spiel, teile den Code und los geht's!
